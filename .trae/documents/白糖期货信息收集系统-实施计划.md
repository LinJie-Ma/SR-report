# 白糖期货信息收集系统 - 实施计划

## 一、项目概述

构建一个白糖期货信息收集与分析系统，后端使用 Python FastAPI，前端使用 Vue 3，实现对白糖期货相关数据的采集、存储、展示。

### 核心目标
- 多维度采集白糖期货相关数据（行情、现货、供需、新闻、天气、ENSO气候）
- 数据存储于 SQLite 数据库
- 通过 Web Dashboard 可视化展示数据
- 定时自动采集，支持每日+盘中多次更新

### ENSO 数据源详细说明

| 来源 | 数据内容 | 获取方式 | 更新频率 |
|------|----------|----------|----------|
| [NOAA CPC](https://www.cpc.ncep.noaa.gov/products/analysis_monitoring/ensostuff/ONI_v5.php) | ONI指数、Niño区域SST距平 | 网页表格解析 + FTP下载 | 每月更新 |
| [NOAA CPC Weekly](https://www.cpc.ncep.noaa.gov/products/analysis_monitoring/lanina/enso_evolution-status-fcsts-web.pdf) | ENSO周报、预测展望 | 网页抓取摘要 | 每周更新 |
| [中国气象局国家气候中心](http://cmdp.ncc-cma.net/cn/monitoring.htm) | ENSO监测简报、NCC判定 | 网页抓取 | 每月更新 |
| [澳大利亚气象局 BOM](http://www.bom.gov.au/climate/enso/) | SOI指数、ENSO展望 | 网页抓取 | 每周更新 |

**采集策略**：每日 10:00 自动检查各数据源最新值，与数据库中已有数据比对，仅在检测到变化时写入新纪录。由于不同数据源更新频率各异（周更/月更），大部分检查日不会有新数据入库，从而避免冗余存储。

### ENSO 数据详解
ENSO (厄尔尼诺-南方涛动) 是影响全球糖料作物（甘蔗/甜菜）产量的关键气候因子：
- **厄尔尼诺 (El Niño)**：通常导致印度、泰国干旱（甘蔗减产），巴西中南部降雨偏多，中国南方可能出现洪涝
- **拉尼娜 (La Niña)**：通常导致巴西中南部干旱（甘蔗减产），东南亚降雨偏多
- **ONI (Oceanic Niño Index)**：NOAA 核心指标，Niño3.4 区域三个月滑动平均 SST 距平 > +0.5°C 为厄尔尼诺，< -0.5°C 为拉尼娜
- **SOI (Southern Oscillation Index)**：塔希提与达尔文气压差标准化值，负值持续指示厄尔尼诺

## 十、关键设计决策

1. **先测试后开发**：每个模块先编写测试用例，再实现功能代码
2. **爬虫容错**：每个采集器独立 try-catch，单个数据源失败不影响其他采集
3. **数据去重**：入库前检查数据是否已存在（按日期+合约/地区+来源去重）；ENSO数据采用"每日检查+变化即写"策略，对比最新值与库内最新值，仅增量写入
4. **来源标注**：所有数据均标注来源，前端通过 DataSourceTag 组件展示
5. **CORS 配置**：开发阶段允许前端 localhost 跨域访问
6. **API 版本化**：使用 `/api/v1/` 前缀，预留升级空间
